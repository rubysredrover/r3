# Ruby's voice clips

All media files in this directory are gitignored.

- Drop Ruby's audio clips here (`.mp3` / `.wav`).
- The cloned-voice training file lives at `clips/ruby_training.mp3` (created by the transcription script).
- Per-file Whisper transcripts live at `clips/transcripts/`.

Voice clips are used only for the demo, with Ruby's family's full consent.
